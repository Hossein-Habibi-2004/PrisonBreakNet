Prison Break Tutorial

Send commands as subject to prison.b.net@gmail.com (capitalize is not matter)

Commands:


  • help

      • Send you the tutorial.

  • mtproto

      • Send you a TXT file of Mtproto proxies (work with "mtproxy" too).

  • config

      • Send you a ZIP file of HTTP-injector and HTTP-Custom configs.

  • v2ray

      • Send you a TXT file of V2ray servers (work with "vless", "vmess" and "trojan" too).

  • apk injector

      • Send you HTTP-injector and V2ray-Plugin APKs download link.

  • apk custom

      • Send you HTTP-Custom APK download link.

  • apk v2ray

      • Send you V2rayNG APK download link.
